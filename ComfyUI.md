# ComfyUI实操

## 安装

```bash
git clone https://github.com/comfyanonymous/ComfyUI.git
cd ComfyUI
git submodule add -f https://github.com/ltdrdata/ComfyUI-Manager.git custom_nodes/ComfyUI-Manager
```

## 运行

```bash
python main.py --listen 0.0.0.0
```
## 模型
